#### 数组定义

`[n]T`是一个有n个类型为T的值的数组.

```
var a [10]int
```

以上表达式的意思是:a是一个有10个整型元素的数组

#### 数组声明

```
var a [5]byte //长度为5的数组，每个元素为一个字节
var b [2*N] struct { x, y int5 } //复杂类型数组
var c [5]*int // 指针数组
var d [2][3]int //二维数组
var e [2][3][4]int //等同于[2]([3]([4]int))
```

#### 数组初始化

先声明再初始化

```
var a [3]string
a = {'1','2','3'}
```
